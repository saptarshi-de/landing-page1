SIGN LANGUAGE INTERPRETER

Watch Video View on GitHub

This app utilizes computer vision and spatial computing to translate real-time American Sign Language (ASL) hand gestures into text. Built with Python, React.js, and YOLOv8, it processes gestures using a custom-trained model and displays results in a highly interactive UI. The project combines backend Python code with a dynamic React frontend, achieving low-latency real-time conversion and advanced UI animations. Key technologies include OpenCV for image processing, and development tools like VSCode and PyCharm. Future goals include expanding functionality to support numerals and enhancing user interaction for a broader range of gestures.

TAMAGOTCHI PET GAME

Watch Video View on GitHub

Created a virtual pet simulation game utilizing object-oriented programming principles, featuring pets from five species with distinct ASCII art. The game leverages polymorphism and inheritance to manage attributes such as energy, hunger, and boredom, which influence the pet's mood. Users can interact with their pets through nine distinct actions—feeding, playing, and grooming—each with three unique methods. The simulation includes a time-based system for realistic pet care and supports saving and loading pet data for persistent gameplay.
